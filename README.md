# PalindromeCheckerApp

## Objective
To develop a Java application that checks whether a given string is a palindrome.

## Project Flow
1. Created GitHub repository PalindromeCheckerApp
2. Cloned repository to local system
3. Created App module in IntelliJ
4. Created PalindromeCheckerApp.java class
5. Implemented palindrome checking logic
6. Used Git commands to track and push code

## Use Case
The program takes a string input and checks whether the string is a palindrome or not.
